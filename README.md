# Masa0221 Tap

Homebrew formulae for tools by [@masa0221](https://github.com/masa0221).

## Installation

Add this tap:

```bash
brew tap masa0221/tap
```

Then install any formula:

```bash
brew install <formula>
```

Or install directly without tapping first:

```bash
brew install masa0221/tap/<formula>
```

## Available Formulae

| Formula | Description |
| ------- | ----------- |
| [aud2txt](https://github.com/masa0221/aud2txt) | Transcribe audio files to text (SRT, TXT, VTT) |
| [eks-admin](https://github.com/masa0221/eks-operations) | Operate EKS cluster |
| [fkds](https://github.com/masa0221/fkds) | Generate speech bubbles in terminal |
| [jclockedio](https://github.com/masa0221/jclockedio) | Clock in/out to Jobcan |
| [mov2mp4](https://github.com/masa0221/mov2mp4) | Convert video files (.mov) to MP4 |
| [oauth2-dev-tool](https://github.com/masa0221/oauth2-dev-tool) | OAuth 2.0 development tool |

## Upgrading

To upgrade a formula:

```bash
brew upgrade <formula>
```

To upgrade all formulae from this tap:

```bash
brew update
brew upgrade
```
